## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/shashwat0cdr/Cpp_dsa.git
```

2. Open the project

```bash
cd Cpp_dsa
```

3. Compile any C++ file

```bash
g++ filename.cpp -o output
```

4. Run

```bash
./output
```

**Windows (MinGW)**

```bash
g++ filename.cpp -o output.exe
output.exe
```
